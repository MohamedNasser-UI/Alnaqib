# Site images

Place your image files here with these **exact** names so the site can display them:

| File name | Use |
|-----------|-----|
| `logo.png` | Brand logo (header) |
| `mystic-orchid.png` | Mystic Orchid product |
| `sultan-afghani.png` | Sultan Afghani product |
| `needs.png` | Needs product |

After adding the files, restart the dev server if it’s running. If images don’t show: restart dev server, then hard refresh (Ctrl+Shift+R). Test: open http://localhost:3000/images/logo.png
