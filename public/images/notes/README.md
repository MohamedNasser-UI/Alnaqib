# Fragrance note images

Small images shown below each product’s main image. Add PNG (or JPG) files with these **exact** names so they display on the product cards:

| File name   | Note   |
|------------|--------|
| `vanilla.png` | Vanilla |
| `tobacco.png` | Tobacco |
| `lemon.png`   | Lemon   |
| `orchid.png`  | Orchid  |
| `oud.png`     | Oud     |
| `amber.png`   | Amber   |
| `rose.png`    | Rose    |
| `saffron.png` | Saffron |
| `cedar.png`   | Cedar   |
| `musk.png`    | Musk    |

Each product uses a subset of these (see `src/lib/products.ts` → `notes`). You can add only the ones you use, or all of them for future products.

Recommended: square or round icons, roughly 64×64 px or larger; they are shown at 32×32 px on the site.
